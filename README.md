# [![doc-img]][doc]

Structured logger implementation using a standard library.

## Install

```sh
go get github.com/unemil/logger
```

**Compatibility:** go >= 1.21

## Levels

- TRACE
- DEBUG
- INFO
- WARN
- ERROR
- FATAL
- PANIC
